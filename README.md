# mJ Update

Repositório de distribuição incremental do mJ Tools.

- `manifest.json` descreve a versão publicada e a integridade de todos os arquivos gerenciados.
- `releases/<versão>/` contém somente arquivos novos ou modificados naquela versão.
- A source permanece no repositório `jandinhoo/mJ-Tools`.

Os arquivos deste repositório são gerados por `UpdateSystem/Build-UpdateRelease.ps1` na source. O script não cria commit nem faz push automaticamente.
